# agent-skills

Roo Code 专用技能库，用于优化不同场景下的 AI 交互体验。

## 📁 Skills 列表

### 🎯 Roo Prompt Optimizer
**路径**: `.roo/skills/roo-prompt-optimizer.md`

专门针对 Roo Code Architect 模式优化的 prompt 技能，适用于：
- 模型：GPT-5.2
- Reasoning：High
- 模式：Architect

**核心功能**：
- 结构化的架构设计 prompt 模板
- 充分利用 high reasoning 能力的提问技巧
- 针对 GPT-5.2 优化的对话策略
- 代码重构、技术选型等场景的最佳实践

## 🚀 快速开始

1. 查看 `.roo/skills/` 目录下的 skill 文件
2. 根据你的使用场景选择合适的 skill
3. 参考模板和示例优化你的 prompt
4. 在 Roo Code 中应用优化后的 prompt

## 💡 使用建议

当使用 **Architect 模式 + GPT-5.2 + High Reasoning** 时：
- ✓ 提供详细的业务背景和技术上下文
- ✓ 要求分步骤推理和多方案对比
- ✓ 明确约束条件和非功能性需求
- ✓ 鼓励深度分析和权衡决策
